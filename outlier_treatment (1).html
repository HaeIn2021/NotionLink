행 삭제 dropna() 값 → NaN

시간 기반 보간 — 단계별 예시 수위(wl) 결측 보간
개요
시간 기반 보간이란?
일반 선형 보간은 행 번호 기준으로 값을 채웁니다.
시간 기반 보간은 실제 시간 간격을 기준으로 값을 채웁니다.
날짜 간격이 일정하지 않은 수위 데이터에 적합합니다.
날짜수위 (m)비고
01-010.37정상
01-02?결측
01-04?결측 (2일 공백)
01-050.45정상
아래 데이터로 단계별로 살펴봅니다.
1단계
시간 간격 확인
01-01 → 01-05 사이 총 시간 간격은 4일입니다.
결측 위치는 01-02 (1일 후), 01-04 (3일 후)입니다.
각 결측값의 시간 비율을 먼저 계산합니다.
전체 간격 = 01-05 − 01-01 = 4일
01-02 비율 = 1일 / 4일 = 0.25
01-04 비율 = 3일 / 4일 = 0.75
2단계
추정값 계산
시작값 0.37m, 종료값 0.45m, 차이는 0.08m입니다.
각 비율에 차이를 곱해 더하면 추정값이 나옵니다.
01-02 추정 = 0.37 + 0.25 × 0.08 = 0.39m
01-04 추정 = 0.37 + 0.75 × 0.08 = 0.43m
선형 보간과 달리 날짜 간격이 고르지 않아도 정확하게 계산됩니다.
3단계
결과 확인
결측값이 채워졌습니다. 추정값은 파란색으로 표시됩니다.
실무에서는 flag 컬럼으로 원본과 추정값을 반드시 구분합니다.
날짜수위 (m)flag
01-010.370 (원본)
01-020.391 (추정)
01-040.431 (추정)
01-050.450 (원본)
