인근 관측소 데이터를 활용한 결측 처리
정상비율법 (Normal Ratio)
인근 관측소의 장기 평균 비율을 이용해 결측값 추정.
기상청 공식 방법으로 빠르고 검증된 기법.
결측값 = (홍천평균 / 인근소평균) × 인근소값
빠름 공식 방법 장기평균 필요
역거리 가중법 (IDW)
가까운 관측소일수록 더 높은 가중치를 부여.
여러 관측소가 있을 때 직관적으로 적용 가능.
가중치 = 1 / 거리²
직관적 관측소 3개+ 거리 정보 필요
크리깅 (Kriging)
"가까운 관측소가 더 비슷하다"는 원리를 수학적으로 정교화한 방법.
단순 거리 가중치가 아니라, 관측소들 사이의 실제 데이터 패턴을 분석해 가중치를 결정합니다.
▶ 작동 원리 보기
1
관측소 간 거리 vs 차이 관계 분석
모든 관측소 쌍을 비교합니다. "춘천-홍천(30km)은 강수량 차이가 작고, 춘천-부산(400km)은 차이가 크다"처럼 거리가 멀수록 값 차이가 커지는 패턴을 먼저 파악합니다.
2
베리오그램(Variogram) 생성
1단계에서 파악한 거리-차이 관계를 그래프로 표현한 것이 베리오그램입니다. 이 곡선이 IDW의 단순 1/거리² 대신 실제 데이터에 맞는 가중치 함수 역할을 합니다.
3
베리오그램으로 가중치 계산 후 예측
결측 지점에서 각 관측소까지의 거리를 베리오그램에 적용해 가중치를 결정하고 값을 추정합니다. IDW보다 정확한 이유가 바로 이 맞춤형 가중치 때문입니다.
